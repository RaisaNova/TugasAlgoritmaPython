Program ini berfungsi untuk menentukan status kelulusan berdasarkan dua nilai ujian yang dimasukkan oleh pengguna. Pengguna diminta untuk memasukkan Nilai A dan Nilai B, kemudian program akan menghitung rata-rata dari kedua nilai tersebut. Jika rata-rata lebih dari 75, maka program menampilkan pesan bahwa pengguna lulus, sedangkan jika 75 atau kurang, program menampilkan bahwa pengguna tidak lulus.Di akhir, program menampilkan seluruh informasi yang meliputi Nilai A, Nilai B, Rata-rata, dan Status Kelulusan.
